# Study OSSP

성균관대학교 **오픈소스 소프트웨어 실습(Open Source Software Practice)** 수업에서 학습한 내용을 정리

## 📚 Learning Topics

* Git 기본 명령어
* Git Branch 전략
* GitHub 활용
* Repository 관리
* Pull Request & Merge
* Linux 환경 설정
* Container 실습
* 오픈소스 프로젝트 이해 및 활용

## 🛠 Tech Stack

* Git
* GitHub
* Linux
* Docker
* Bash Shell


## 🎯 Purpose

오픈소스 소프트웨어 개발 환경에 대한 이해를 높이고, Git 기반 협업 및 버전 관리 역량을 향상시키는 것을 목표로 합니다.
